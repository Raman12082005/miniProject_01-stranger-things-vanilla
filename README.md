# Stranger Things - Vanilla JS

A Stranger Things themed landing page built with pure HTML, CSS, and JavaScript.

## Features
- Mouse reveal fire effect
- Smooth cursor tracking with lerp animation
- Entrance animations with CSS keyframes
- Responsive navbar with hover effects
